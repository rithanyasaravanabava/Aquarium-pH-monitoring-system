Title: Aqua pH monitoring system


 Aquarium pH Monitor:

A Python-based simulation of an aquarium pH monitoring system that generates random pH values between 6.0 and 8.5, evaluates whether they fall within the safe range of 6.8 to 8.0 for aquatic life, and provides specific recommendations for maintaining optimal water conditions.

Key functionality:
- Random pH generation within realistic aquarium ranges
- Safety status determination (Safe: 6.8-8.0, Unsafe: outside range)
- Actionable recommendations:
  * Low pH (< 6.8): Add pH UP chemicals
  * High pH (> 8.0): Add pH DOWN chemicals or perform water change
  * Safe range: Maintain current conditions

Run with: `python aquarium_ph_monitor.py`

Ideal for demonstrating conditional logic, formatted output, and real-world environmental monitoring applications in basic programming.
